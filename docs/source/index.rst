系统概述
--------

AI PPT Assistant是一个强大的AI驱动的演示文稿生成系统，具有以下特性：

* **智能内容生成**: 基于Amazon Bedrock AI模型自动生成演示文稿内容
* **多语言支持**: 支持中文、英文等多种语言
* **灵活的样式**: 提供专业、创意、学术等多种演示风格
* **自动图片搜索**: 智能匹配和添加相关图片
* **演讲者备注**: 自动生成详细的演讲备注
* **云原生架构**: 基于AWS Lambda的无服务器架构
